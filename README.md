# GestBiblioteca — Sistema de Gestion de Prestamos
## ASP.NET Core MVC

## Instrucciones de ejecucion

1. Clonar el repositorio:
   git clone https://github.com/TU_USUARIO/GestBiblioteca

2. Entrar a la carpeta del proyecto:
   cd GestBibliotecaApp

3. Ejecutar el proyecto:
   dotnet run

4. Abrir en el navegador:
   http://localhost:5000/Prestamo

## Nota
Las funcionalidades extendidas (Delete con confirmacion, CSRF y validacion
cliente) se desarrollaron en la rama feature/extension-delete-csrf e
integradas a main mediante Pull Request.
